# Object Literal

[Object Literal](./../assets/object-literal.pdf)

## Object Literal Behaviour

- Menambahkan number atau string sebagai key.

  ```javascript
  let a = {};
  a[1] = "this is 1";
  a[0] = "this is 0";
  console.log(a);
  
  let b = {};
  b["c"] = "this is c";
  b["a"] = "this is a";
  console.log(b);
  
  let c = {};
  c[20] = "this is 20";
  c[10020] = "this is 10020";
  c["c"] = "this is c";
  c["a"] = "this is a";
  console.log(c);
  ```

- Menambahkan key yang sudah ada sebelumnya.

  ```javascript
  let a = {
    key1: 'String',
    key2: 10,
    key3: true
  };
  console.log(a);
  a.key1 = 99;
  console.log(a);
  ```
